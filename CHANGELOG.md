# Changelog

Все заметные изменения в проекте будут задокументированы в этом файле.

## [2.0.0] - 2026-01-19

### Исправлено

- **Критическая ошибка**: Исправлена ошибка "WG_CLIENT_PRIVATE_KEY: unbound variable", которая происходила при наличии пустых значений в файле .env
- **Проверка переменных**: Добавлена проверка на определенность всех критических переменных в функциях `prepare_configs()` и `show_connection_info()`
- **Генерация параметров**: Обновлена функция `generate_secrets()` для корректной перегенерации недостающих параметров при наличии пустых значений в .env

### Добавлено

- **Файл шаблона**: Создан .env.template с пустыми значениями для безопасной инициализации конфигурации
- **Скрипт проверки**: Добавлен health-check.sh для проверки работоспособности VPN-сервера
- **Документация**: Созданы файлы README.md, docs/qwen.md и docs/roadmap.md
- **Лицензия**: Добавлен файл LICENSE с текстом MIT-лицензии
- **Безопасность**: Добавлен .gitignore для исключения конфиденциальных файлов из репозитория

### Изменено

- **Безопасность**: Обновлены проверки для предотвращения использования пустых или неопределенных значений
- **Структура проекта**: Улучшена организация файлов и документации

## [1.0.0] - 2024-XX-XX

### Добавлено

- Первоначальная реализация мульти-протокольного VPN-сервера
- Поддержка протоколов VLESS+Reality, Shadowsocks-2022 и AmneziaWG
- Автоматическая генерация криптографических параметров
- Интеграция с Docker и Docker Compose
- Автоматическая настройка брандмауэра UFW
- Генерация QR-кодов для подключения
