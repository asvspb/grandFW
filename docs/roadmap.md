Ты — эксперт по сетевой безопасности, системному администрированию Linux и Docker.

Твоя задача: Написать полное, готовое к развертыванию решение (код и скрипты) для моего VPS сервера (Ubuntu 22.04/24.04), которое поднимет 3 современных VPN/Proxy протокола, устойчивых к DPI-блокировкам.

**Требования к протоколам:**
1. **VLESS + XTLS-Reality** (на базе ядра Xray-core).
2. **Shadowsocks-2022** (с использованием AEAD-2022 шифрования, тоже через Xray).
3. **AmneziaWG** (форк WireGuard с защитой от детекции).

**Технические требования к реализации:**
1. Использовать **Docker** и **Docker Compose** для изоляции сервисов.
2. Написать bash-скрипт `install.sh`, который сделает всё автоматически:
   - Проверит и установит Docker и Docker Compose.
   - Сгенерирует все необходимые ключи (UUID для VLESS, пары ключей x25519 для Reality, пароли для Shadowsocks, ключи для AmneziaWG).
   - Создаст файлы конфигурации (`config.json` для Xray и конфиги для Amnezia).
   - Подставит сгенерированные ключи в конфиги.
   - Запустит контейнеры.
3. **Важно:** Скрипт в конце работы должен вывести в консоль готовые строки подключения (Links) или данные для клиентов:
   - Ссылку формата `vless://...`
   - Ссылку формата `ss://...`
   - Конфиг для AmneziaWG (текстом, который можно скопировать в файл .conf).

**Детали настройки:**
- Для Reality используй маскировку под легитимный сайт (например, `www.microsoft.com` или `dl.google.com`).
- Порт для VLESS/Reality должен быть 443.
- AmneziaWG должна использовать свои специфические параметры (Jc, Jmin, Jmax и т.д.) для обфускации пакетов.

Пожалуйста, предоставь:
1. Файл `docker-compose.yml`.
2. Шаблоны конфигурационных файлов.
3. Главный скрипт `install.sh` с комментариями, объясняющими, что делает каждая строка.
